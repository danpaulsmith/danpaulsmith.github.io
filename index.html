.prototype.

When you are using prototypal inheritence, you should always call the super constructor from the sub-class constructor to make sure that your sub-class instance gets a unique copy of all properties, simple and complex. By calling the super constructor with the sub-class context, we force the prototype to copy all default values into the instance of our sub-class.
